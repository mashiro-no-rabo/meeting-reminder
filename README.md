# MeetingReminder

A macOS menu bar app that shows a full-screen overlay 2 minutes before your next calendar event.

## Features

- Menu bar icon with manual "Show Overlay" trigger
- Polls calendar every 30 seconds via EventKit
- Full-screen semi-transparent overlay on all screens
- Dismiss by clicking, pressing ESC/Enter, or wait 30s for auto-dismiss
- Runs as a launchd agent on login

## Install

Use `/install` in Claude Code.
